## Welcome to Asset Laboratory Professional

Asset Laboratory Professional is a comprehensive asset management platform designed for modern enterprises. Built with cutting-edge technologies including Next.js 15, React 19, and HeroUI Pro, it provides a robust foundation for tracking, managing, and analyzing your organization's assets.

## Quick Start

Get up and running with Asset Laboratory Professional in minutes:

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/Chrisdebug123/Asset_Laboratory_professional
    cd Asset_Laboratory_professional
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Setup Environment">
    Configure Doppler for secure environment management:
    ```bash
    doppler setup
    doppler login
    ```
  </Step>

  <Step title="Start Development">
    ```bash
    pnpm dev
    ```
    Your application will be available at:
    - Frontend: http://localhost:3000
    - Backend API: http://localhost:3001
  </Step>
</Steps>

## Architecture Overview

Asset Laboratory Professional follows a monorepo architecture using pnpm workspaces:

```
packages/
├── client/    # Next.js frontend application
├── server/    # Express.js backend API
└── shared/    # Shared types and utilities
```

## Technology Stack

### Frontend
- **Framework**: Next.js 15.5 with React 19
- **UI Library**: HeroUI Pro 2.8.3
- **Styling**: Tailwind CSS 4.1
- **State Management**: React Context & Hooks
- **Theme**: next-themes with dark/light/system modes

### Backend
- **Runtime**: Node.js 22+ with Express 5.1
- **Language**: TypeScript 5.5
- **Database**: PostgreSQL with Prisma 6.11.1
- **Authentication**: JWT-based auth
- **Validation**: Zod schema validation

### DevOps & Tools
- **Package Manager**: pnpm 9.15.2 with workspaces
- **Build Tool**: Turbopack
- **Secrets Management**: Doppler CLI 3.75.1
- **Version Control**: Git with GitHub CLI
- **CI/CD**: GitHub Actions with Dependabot
